# JavaScript Mini Projects

A collection of beginner-friendly JavaScript projects that I built while learning JavaScript. These projects helped me understand DOM manipulation, event handling, user interaction, and core JavaScript concepts through hands-on practice.

## Projects Included

### 01 - Color Change
A simple application that changes the background color using JavaScript.

**Concepts Used:**
- DOM Manipulation
- Event Listeners
- CSS Styling with JavaScript

---

### 02 - BMI Calculator
A BMI (Body Mass Index) calculator where users can enter their height and weight to calculate their BMI.

**Concepts Used:**
- Form Handling
- Input Validation
- Mathematical Calculations
- Conditional Statements

---

### 03 - Digital Clock
A real-time digital clock that displays the current time and updates every second.

**Concepts Used:**
- Date Object
- `setInterval()`
- DOM Updates

---

### 04 - Guess The Number
A fun number guessing game where the user tries to guess a randomly generated number within limited attempts.

**Concepts Used:**
- Random Number Generation
- Loops & Conditions
- User Input Handling
- Game Logic

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

The goal of this repository is to strengthen my JavaScript fundamentals by building small, practical projects before moving on to advanced topics and full-stack development.
